# 使用ElementPlus

如果能直接使用elementPlus，那多是一件美事啊！

## 1、npm安装



## 2、引入

在 `/docs/.vitepress/theme` 目录下新建一个`index.js`文件。

```js
import ElementPlus from 'element-plus'
import 'element-plus/dist/index.css'

import theme from 'vitepress/dist/client/theme-default/index'

export default {
  ...theme,
  enhanceApp ({ app }) {
    app.use(ElementPlus)
  }
}
```
