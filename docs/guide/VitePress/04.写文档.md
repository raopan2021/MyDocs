# 文档页面

在写文档时，一般在`markdon`文件最上方注明页面类型:`Page` /`Doc` 

它决定了文档在页面的布局、各种默认样式。从表现形式上看也只是 **`文章内容显示位置上面的不同`**

大多数情况下在编写内容时推荐使用 `doc`!

 `page` 相比较 `doc` 会默认缺少一些内容比如 `Edit Link`，`NextPage` `PrePage` 等。



```markdown
---
layout: doc
---
巴啦啦能量
```

```markdown
---
layout: page
---
真的吗？我不信！
```

接下来就开始写文档吧。。。
